GET

GET Method

Sends data via URL parameters. Ideal for search queries and read-only actions. Data is visible in the browser URL — never use for sensitive information.

GET /search?query=html&page=1
POST

POST Method

Sends data in the request body, hidden from the URL. Used for form submissions, login, and any action that creates or modifies data on the server.

body: { name, email, message }
