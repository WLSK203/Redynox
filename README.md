# Redynox Codes Repository

Welcome to the Redynox Codes repository. This repository contains the source code for various tasks and projects developed under the Redynox Web Development Internship.

## Project Structure

---

### [Task 1: Redynox Landing Page](./task-1)

A modern, responsive, and pixel-perfect landing page for the Redynox Web Development Studio.

**Technologies Used:** `HTML5`, `CSS3`, `JavaScript`

**Features:**

- Fully responsive design following a Mobile-first approach.
- Modern web design aesthetics including glow effects, gradients, and a sleek user interface.
- Terminal-like visual animation and dynamic statistics.
- Interactive navigation bar with a hamburger menu for mobile devices.
- Optimized for performance, semantic structure, and SEO best practices.

**Usage:**

- Open `task-1/index.html` in your web browser.

---

### [Task 2: JavaScript Form Validation](./task-2)

A fully-featured contact form demonstrating advanced client-side JavaScript validation, real-time feedback, and dynamic user interfaces.

**Technologies Used:** `HTML5`, `CSS3`, `JavaScript`

**Features:**

- Real-time input validation using Regex (for names, emails, and message rules).
- Live error states providing instant feedback on blur and input, with accessibility enhancements (`aria-live`, `role="alert"`).
- Dynamic character counting for the message textarea.
- Simulated asynchronous form submission representing a POST request with a loading state.
- Dynamic success output overlay that renders submitted data to the UI.
- An educational aside explaining the concepts demonstrated and the difference between GET vs POST HTTP methods.

**Usage:**

- Open `task-2/form.html` in your web browser.

---

## Getting Started

To view any of the tasks locally:

1. Clone this repository or download the source code files.
2. Navigate to the specific task directory (e.g., `cd task-1`).
3. Open the main HTML file (`index.html` for Task 1, `form.html` for Task 2) in your preferred modern web browser.
4. No build tools or package installations are required as these are vanilla web projects.
